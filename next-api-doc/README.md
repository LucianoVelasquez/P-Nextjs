# Proyecto: Registro y Login de Usuarios con React y Next.js 14

Este proyecto consiste en una aplicación web sencilla desarrollada con React y Next.js 14. La aplicación permite a los usuarios registrarse y loguearse, utilizando diversas tecnologías para la gestión de la interfaz, autenticación, base de datos y estilo.

## Getting Started

First, run the development server:

```bash
npm i

npm run dev

```

## .env
Crear el archivo .env en la raiz del proyecto y agregar la variable de entorno con la url de la DB PostgreSQL.

```bash
DATABASE_URL = "..."

```

## Tecnologías utilizadas

 * ShadCN UI: Framework para la construcción de interfaces de usuario de alta calidad.
 * NextAuth: Biblioteca para la gestión de la autenticación en aplicaciones Next.js.
 * Tailwind CSS: Framework de utilidades CSS para estilizar la interfaz de usuario de manera rápida y       eficiente.
 * Prisma: ORM (Object-Relational Mapping) para trabajar con la base de datos de manera sencilla y segura.
 * PostgreSQL: Sistema de gestión de bases de datos relacional utilizado para almacenar la información de los usuarios.
